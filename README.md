# Python-for-network-engineers
This repository contains learning materials on python for network engineers
\n
The python files are scripts made executable in linux.\n
Steps to make the files executable in linux:\n
  example:\n
          -create and edit a file with 'vi' -text editor\n
          -$vi example.py ->> creates the file and enters the editor\n
          -while in text editor:\n
               -command 'i' to enable editing\n
               -sample text;\n
                                #!/usr/bin/env python\n
                                print(f"Enter your name:{input()}")\n
               -then: esc + :wq ->>quits and saves the input\n
          $chmod 755 example.py ->>changes the file to executable\n
          $./example.py  ->>executes the file\n
